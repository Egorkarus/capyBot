# capyBot 🍊

Самодовольная капибара-музыкальный бот для вашего Discord-сервера. Она ленива, харизматична и любит слушать музыку, пока греется на солнышке.

Бот умеет воспроизводить музыку из YouTube, SoundCloud, по прямым ссылкам на MP3-файлы, а также проигрывать MP3-файлы, прикрепленные напрямую к сообщениям в Discord.

---

## 🚀 Команды

Префикс бота: `!`

### Для всех пользователей
* **`!play <ссылка/файл>`** — Воспроизвести музыку. Поддерживаются ссылки на YouTube, SoundCloud, прямые ссылки на `.mp3` или прикрепленный к сообщению MP3-файл. Если трек уже играет, добавляет его в очередь.
* **`!skip`** — Пропустить текущую песню и перейти к следующей в очереди.
* **`!pause`** — Поставить воспроизведение на паузу.
* **`!resume`** — Снять воспроизведение с паузы.
* **`!queue`** — Показать список следующих треков в очереди.
* **`!help`** — Показать справочное сообщение с командами бота.

### Только для Администраторов (👑)
* **`!join`** — Пригласить капибару в ваш текущий голосовой канал.
* **`!leave`** — Выгнать капибару из голосового канала.
* **`!stop`** — Остановить воспроизведение музыки и полностью очистить очередь.

## 🛠️ Настройка окружения

Создайте файл `.env` в корневой директории проекта и добавьте ваш токен Discord бота:

```env
DISCORD_TOKEN=ваш_токен_discord_бота
```

## 📦 Запуск бота

### Вариант 1. Запуск через Docker (Рекомендуемый)

Убедитесь, что у вас установлены **Docker** и **Docker Compose**. Бот автоматически использует утилиту `yt-dlp` внутри контейнера для скачивания аудио.

Запуск контейнера в фоновом режиме:
```bash
docker-compose up -d --build
```

Остановка контейнера:
```bash
docker-compose down
```

### Вариант 2. Локальный запуск (Node.js)

Для локального запуска вам понадобятся:
* **Node.js** версии 20+
* **ffmpeg** (должен быть установлен в вашей операционной системе и добавлен в PATH)

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите бота:
   ```bash
   npm start
   ```

## 🛠️ Архитектура и технологии
* **discord.js v14** — библиотека для работы с API Discord.
* **@discordjs/voice** — современный движок для работы со звуком в Discord голосовых каналах.
* **yt-dlp** — скачивание аудио-потоков.
* **ffmpeg** — декодирование и обработка звука.
